 # Electric Vehicle Population Analysis using Power BI

## Project Overview
This project focuses on analyzing Electric Vehicle (EV) population data using Microsoft Power BI. The aim is to convert raw data into meaningful insights through interactive dashboards and visualizations.

Electric vehicles are becoming an important part of modern transportation due to environmental concerns and rising fuel costs. This project helps in understanding EV adoption trends and patterns.

## Objectives
- Analyze growth of electric vehicles over time
- Study distribution of EV types (BEV and PHEV)
- Identify top manufacturers
- Analyze electric range of vehicles
- Study geographical distribution of EVs

## Tools and Technologies
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- CSV Dataset
- Data Cleaning and ETL Process

## Dataset
The dataset contains information about:
- Vehicle Type
- Manufacturer
- Model
- Electric Range
- Registration Year
- Location

## Key Insights
- EV adoption is increasing every year
- BEVs are more popular than PHEVs
- Urban areas have higher EV usage
- Electric range is improving with new models

## Dashboard Features
- Interactive charts and graphs
- Filters and slicers
- Trend analysis
- Easy-to-understand visuals

## Future Scope
- Integration of real-time data
- Machine learning prediction models
- Charging station analysis
- Smart city implementation

## Conclusion
This project demonstrates how Power BI can be used to analyze electric vehicle data and generate useful insights. It helps in understanding the growth and future of EV adoption.
